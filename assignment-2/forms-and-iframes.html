Forms and Iframes

An Iframe (inline frame) is used to embed another document within the current HTML document.

It's used with title, source attributes, together with height and width properties.

A good use case for Iframes is to display a web page within a web page. With examples such as Social media profiles, Advertisements, videos (like YouTube), and Web analytics.

            <iframe src="url" title="description" height="" width=""></iframe>
