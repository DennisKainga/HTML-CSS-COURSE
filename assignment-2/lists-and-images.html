Imaging Mapping

What is it, and how to use it

HTML Image Maps

An image map allows parts of an image to be clickable. It is created using the <map> tag with clickable areas defined by <area> tags.

The Image

Use the <img> tag to insert the image. Add the usemap attribute to link it to an image map:

    <img src="imagename.jpg" alt="imagename" usemap="#mapname">

The Map

Create an image map with the <map> element. Set its name attribute to match the image's usemap attribute value:

    <map name="mapname"></map>

The Clickable Areas

Define clickable areas with <area> tags. Specify the shape and coordinates:

    <area shape="rect" coords="30,50,140,160" href="#">

This example defines a rectangular area starting at (30, 50) and ending at (140, 160) pixels.
